# Guardrails README

Guardrails are ai-driven and consistency-orientated. They exist to detail the crucial rules to follow when performing a
specific task

> NOTE:
> Likely **NOT** able to be invoked at the beginning of a chat by using the slash `/`

## Structure

- Similar structure and recommendations to [Prompts](../prompts/README.md#structure), however may not show in the chat
  when invoked via `/` (slash) at the beginning of a chat
- Uses a prefix of `when` in the filename to remind the reader that this is meant to be ai-driven
